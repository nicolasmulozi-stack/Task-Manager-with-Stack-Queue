# 📋 Task Manager

## 📌 Description
A task management system that allows users to add, process, and manage tasks using basic data structures.

## ⚙️ Features
- Add and manage tasks
- Process tasks in order
- Simple task tracking system

## 🧠 Concepts Used
- Data structures (Queue/Stack)
- Lists (ArrayList)
- Control structures

## ▶️ How to Run
1. Compile:
   javac TaskManager.java
2. Run:
   java TaskManager
